# 🌱 Smart Terrace Farming Monitoring System

An IoT-based Smart Terrace Farming Monitoring System that monitors environmental conditions using sensors and displays real-time data in an Android application.

---

## 📌 Project Overview

This project was developed as a Mini Project during my second year of B.Tech Information Technology.

The system uses IoT sensors to monitor:

- 🌡 Temperature
- 💧 Humidity
- 🌱 Soil Moisture

The collected sensor data is transmitted through NodeMCU ESP8266 and displayed in an Android application developed using Java and XML.

---

## 🚀 Features

- User Registration
